here we are saying that all elements with p tag should have one font style and all elements with label tag should have size 10px,
<head> <style> *{ font-size:10px; } label{ font-size:20px; } </style> </head> <body> <p> hi i am a paragraph </p> <label> hello </label> </body>
output is :

hi i am a paragraph



here you have a doubt label element is large in font because we have written label tag should be 20px in css but do you know why p element is small in size
because the reason we have written *{property} means default property means we haven't assigned any property for p so p will have a default property, but for label we have assigned a property so label elements will not read default property
<head> <style> *{ font-size:10px; /* i am a comment */ } </style> </head> <body> <p> hi i am a paragraph </p> </body>
