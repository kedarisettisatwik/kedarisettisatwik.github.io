1. history of computer's ..

presently we are using 5th generation computer's.

In first generation (1946 - 1959) :
computers are manufactured using thousands of vaccum tubes. which are made up of fragile glass.
programming is done by using maching learning.
examples : ENIAC, EDVAC, EDSAC, IBM701

In second generation (1959 - 1965):
usage of transistors, high development programming languages started.
examples : IBM 7030, CDC 1604, HONEY WELL 400

In third generation (1965 - 1971) :
invention of integrated chips ( IC )which consists of thousands of transistors, capacitors, resistors on a single chip.
examples :IBM 360/370, CDC 6600, PDP - 8

SSI - small scale IC (10 -20 components)
LSI - large scale IC (30,000 components)
VLSI - very large scale IC (1Million components)
ULSI - ultra scale IC (10Million components)

In fourth generation (1975 - 1980) :
using of LSI, VSLI, Microcomputers came.
Java, Python, C++, C are developed
examples : IBM PC , TRS-80, CRAY - 1,2,X/MP

In fifth generation (1980 - present) :
using of ULSI came.
examples : PENTIUM PC'S

2. hard & soft ware

we should know about some of the elements

DATA - all computer processors require data, which is a collection of raw facts, symbols, numbers, character's etc..

INFORMATION - a data is converted to information so that it can answer who what why when type of questions.

example: in our general sense : 94 is a data , i got 94 marks in exam is information.

INPUT - computer will work based on the data,instructions given by user, the data that we give using keyboards, remote, mouse, touch etc.. are know as inputs.

OUTPUT - the data present in screen we see is output.

STORAGE UNIT - data has to be feed into system before actual processing starts.

PROCESSOR - performs the task based on our instructions.(CPU)

CONTROL UNIT - controlls the data traffic and flow of information between hardware and software components.

ALU - Arthematic Logical Unit, the place where math calculations takes plaace.

1 BYTE = 8 BITS
bit means a box in which comoputer stores 1 or 0

3. programming..

generally we divide language into 2 types HIGH LEVEL LANGUAGE , LOW LEVEL LANGUAGE,

high level means java,python,programming languages,
low level means full of 0's and 1's only machines can read this..

the process of converting high level to low level is done by compiler , interpetter.

COMPILER - means reads the full code that we write at a time, if full code is right then only it will show the output of our code,

INTERPETTER - means reads one line after one line in code that we write, if there is wrong in code then the portion above that wrong will be executed..
