1. history of computer's ..

presently we are using 5th generation computer's.

In first generation (1946 - 1959) :
computers are manufactured using thousands of vaccum tubes. which are made up of fragile glass.
programming is done by using maching learning.
examples : ENIAC, EDVAC, EDSAC, IBM701

In second generation (1959 - 1965):
usage of transistors, high development programming languages started.
examples : IBM 7030, CDC 1604, HONEY WELL 400

In third generation (1965 - 1971) :
invention of integrated chips ( IC )which consists of thousands of transistors, capacitors, resistors on a single chip.
examples :IBM 360/370, CDC 6600, PDP - 8

SSI - small scale IC (10 -20 components)
LSI - large scale IC (30,000 components)
VLSI - very large scale IC (1Million components)
ULSI - ultra scale IC (10Million components)

In fourth generation (1975 - 1980) :
using of LSI, VSLI, Microcomputers came.
Java, Python, C++, C are developed
examples : IBM PC , TRS-80, CRAY - 1,2,X/MP

In fifth generation (1980 - present) :
using of ULSI came.
examples : PENTIUM PC'S

19. break & continue

break = stop
continue = skip
break, continue are keywords used on loops..

while(){ } for (){ } you loops execute the code with in { } repeatly until the condition is false.. now lets see how break, continue works..

for (int i = 0; i < 5; i++){ printf(" %d ", i); } for (int i = 0; i < 5; i++){ if (i == 3){ break ; } printf(" %d ", i); } for (int i = 0; i < 5; i++){ if (i == 3){ continue ; } printf(" %d ", i); } output of 1st code is 0 1 2 3 4 output of 2nd code is 0 1 2 output of 3rd code is 0 1 2 4 because, in 2nd code we wrote if i == 3 break means stop reading code and go outside of code... because, in 3rd code we wrote if i == 3 continue means skip this step and go to next step means i = 4 this applies for while loop also, try it yourself..
